Google Sheets y Apps Script
- JavaScript - Obtener el historial de tipos de cambio desde una API en Google Sheets.
- JavaScript - Agregar automáticamente el tipo de cambio de cierre de ayer cada mañana (Trigger).
- JavaScript - Monitorear cambios cuando el tipo de cambio aumenta 1 % o más respecto al tipo de cambio actual en uso.
- JavaScript - Enviar un correo con un resumen automáticamente por la mañana cuando hay cambios (Trigger).


Power BI
- TipoCambio_Mensual - Tipo de cambio del último día del mes pasado (estrategia actual).
- TipoCambio_Aumento - Aumento del tipo de cambio solamente.
- Precio_dolares - Precio según la estrategia actual.
- Precio_en_Dolares_Diario - Precio según el tipo de cambio diario.
- Precio_en_Dolares_aumento - Precio según el aumento del tipo de cambio.
